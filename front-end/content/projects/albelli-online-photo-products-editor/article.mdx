## Type

Employment at Albelli

## Description

It is an online photo editor for photo books, calendars, wall decor, mugs etc. One of multiple client applications for editing products with custom design.

## Technologies

- JavaScript
- Google Closure Library
- React
- Node.js
- SCSS
- Google Closure Compiler
- Webpack

## Main features

- Online editor of photo books, calendars, cards, mugs and wall decorations
- Performant rendering on Canvas
- Failure resistant photo uploading
- Photo effects, resizing and rotation
